# 🌐 Jeremias Panozzo - Personal Website

Este es mi sitio web personal desarrollado con [Astro](https://astro.build/). El sitio presenta mi perfil profesional, proyectos destacados, artículos y enlaces a redes sociales. Está desplegado en GitHub Pages.

## 🚀 Tecnologías utilizadas

- [Astro](https://astro.build/) – Framework web ligero
- [TypeScript](https://www.typescriptlang.org/) – Tipado estático
- [Tailwind CSS](https://tailwindcss.com/) – Estilos utilitarios
- [GitHub Pages](https://pages.github.com/) – Hostings
